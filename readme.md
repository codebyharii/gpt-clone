# ChatGPT UI Clone

A single-page clone of the ChatGPT interface built with plain HTML, CSS, and JavaScript.

## Features

- Dark sidebar with navigation and user avatar
- Landing page with "What are you working on?" prompt
- Suggestion chips for quick actions
- Chat mode with user and AI message bubbles
- Typewriter streaming effect for AI responses
- Auto-resizing textarea input
- New chat button to reset the conversation

## How to Use

1. Download `gpt-search-clone.html`
2. Open it in any modern browser
3. Type a message or click a suggestion chip to start chatting

## Tech Stack

- **HTML5** — structure
- **CSS3** — layout, animations, dark theme
- **Vanilla JavaScript** — chat logic, typewriter effect, state management

## Notes

- No dependencies or build tools required
- Responses are mocked locally — not connected to any API
- To connect a real AI, replace the `getResponse()` function with an API call
